# 003. Claude Haiku como modelo default

## Estado

Aceptada.

## Contexto

Las extracciones documentales necesitan costo bajo y latencia razonable.

## Decision

Usar Claude Haiku 4.5 como default y Sonnet 4.5 solo como fallback o para analisis de riesgo contractual.

## Consecuencias

El costo variable baja y los pasos de razonamiento complejo quedan acotados.
